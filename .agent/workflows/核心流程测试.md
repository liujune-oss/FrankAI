## 工作流说明

此工作流用于自动化测试**日常活动（任务/日程/日志）生成的全链路核心流程**。基于你的需求，流程完全闭环：
- 由模型全自动生产复杂的自然语言边界进行测试
- 二度语义模型分析
- Json 反射取值
- 代码级白盒断言验证字段不可为空且可有效写入
- 真正的 Supabase DB 写入

全部过程将在命令行通过自动化 Node.js 脚本分步执行且将所有输入、分析输出、提取 JSON、校验和实际写入成败日志记录入文档中。无需任何人工浏览器刷新介入。

### 步骤清单

1. 在终端验证目录存在并直接执行自动化验证脚本：
```powershell
mkdir -p scripts && cd f:\FrankAI\gemini-chat-pwa && npx tsx scripts/core-flow-test.ts
```

2. 打开并查阅自动更新的日志文件： `f:\FrankAI\gemini-chat-pwa\core_flow_test_log.md`
该日志将会作为后续 AI 修错排查的翔实依据，其中不仅会包含十轮成功的样本，也会用标注符号显式指出报错被重试拦截的跳过环节。
