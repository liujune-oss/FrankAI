# Gemini Chat — 产品介绍

## 概述

Gemini Chat 是一个由 Google Gemini 驱动的私人 AI 聊天助手，以 PWA（渐进式 Web 应用）形式交付，可安装到手机或桌面，提供接近原生应用的体验。

## 核心功能

### 💬 智能对话
- 基于 Gemini 3 Flash / Pro 模型的流式对话
- 支持 Markdown 渲染（表格、列表、代码块等）
- 显示模型思考过程（Thinking），可折叠查看
- 内置 Google Search 工具，可实时联网搜索

### 🎨 AI 图片生成
- 智能识别绘图意图（中英文关键词匹配）
- 基于 Gemini 2.5 Flash Image 模型生成图片
- 支持连续对话式图片编辑（上下文保持）

### 📷 图片理解
- 支持上传图片发送给 AI 分析
- 多图同时上传

### 📝 会话管理
- 多会话支持，侧栏抽屉式会话列表
- 自动标题生成（取用户首条消息）
- 单条删除 / 全部清空
- 数据本地存储（IndexedDB），不依赖服务器

### ⚙️ 自定义系统指令
- 可自定义 AI 的行为角色和回复风格
- 内置默认指令：鼓励批判性思维，拒绝无脑顺从

### 🔐 账号与管理员系统
- 专属 Admin 控制台生成 4 位纯字母邀请码
- 激活码 + 设备指纹 + JWT 鉴权
- 一键吊销多设备权限，防止滥用

### 🧠 持续记忆 (Continuous Memory)
- 支持提取独立会话中的关键事实与设定
- 利用 gemini-embedding-001 模型转化为 3072 维记忆向量存入云端 (Supabase)
- 新对话中自动采用 RAG (检索增强) 无感调取相似记忆
- 跨设备云同步自定义系统指令 (System Instructions)

### 📱 PWA 体验
- 可安装到主屏幕，全屏独立运行
- Service Worker 离线缓存
- 跟随系统深色/浅色模式切换
- 安全区域适配（iPhone 刘海屏等）

### 🎨 模型选择
- 支持在界面切换不同 Gemini 模型
- 默认使用 `gemini-3-flash-preview`
